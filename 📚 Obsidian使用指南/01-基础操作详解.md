# 🛠️ Obsidian 基础操作详细指南

## 🎯 界面布局详解

### 左侧边栏（默认从顶到底）：
1.  **文件浏览器**：查看本地Vault中的所有文件和文件夹
2.  **搜索**：在所有笔记中搜索关键词或正则表达式
3.  **反向链接**：查看当前笔记被哪些其他笔记引用
4.  **标签**：按标签分类查看所有笔记
5.  **大纲**：显示当前笔记的标题结构
6.  **命令面板**：通过快捷键`Ctrl+P`/`Cmd+P`打开，执行所有操作

### 主编辑区：
- **编辑模式**：默认的纯文本编辑模式，支持Markdown
- **预览模式**：点击右上角眼睛图标，渲染为最终效果
- **源模式**：显示完整的Markdown源代码（包括不可见的格式）

### 右侧边栏：
- 显示当前笔记的反向链接、标签、大纲等信息

## 📝 新建与编辑笔记

### 方法1：通过文件浏览器
1.  点击文件浏览器右上角的「+」图标
2.  输入笔记名称（无需加`.md`后缀，Obsidian会自动添加）
3.  回车确认，即可在编辑区打开新笔记

### 方法2：通过双向链接快速创建
1.  在任意笔记中输入`[[新笔记名称]]`
2.  按住`Ctrl`点击这个链接，会提示「创建新笔记」
3.  点击创建后，自动打开新笔记并跳转

### 编辑技巧：
- 快捷键`Ctrl+S`/`Cmd+S`：保存笔记（Obsidian会自动保存，但手动保存更稳妥）
- 快捷键`Ctrl+Z`/`Cmd+Z`：撤销操作
- 快捷键`Ctrl+Y`/`Cmd+Y`：重做操作

## 🔗 双向链接进阶

### 别名链接：如果笔记名称太长，可以使用别名
```markdown
[[我的第一篇非常长的笔记名称|短别名]]
```

### 链接到文件夹：
```markdown
[[📚 Obsidian使用指南/]]
```

### 链接到特定标题：
```markdown
[[00-快速入门总览#双向链接基础]]
```

### 取消链接：
如果想把`[[链接]]`变回普通文本，可以在前面加反斜杠：`​[[链接]]`（实际输入时去掉中间的空格）

## 🏷️ 标签系统

### 基本用法：
在笔记中任意位置输入`#标签名`，比如：
`#学习` `#Obsidian` `#技术笔记`

### 标签规范：
- 不要使用空格，使用`-`或`_`代替：`#obsidian-tutorial` 而不是`#obsidian tutorial`
- 可以使用层级标签：`#学习/Obsidian`

### 查看所有标签：
点击左侧边栏的「标签」图标，即可看到Vault中所有使用过的标签

---

## 🔍 搜索功能

### 基础搜索：
在搜索框输入关键词，即可在所有笔记中查找包含该关键词的内容

### 高级搜索：
点击搜索框右侧的「放大镜」图标，进入高级搜索模式，可以使用：
- 正则表达式
- 排除特定关键词：`-关键词`
- 搜索特定文件夹：`folder:📚 Obsidian使用指南`
- 搜索特定标签：`tag:#Obsidian`

## 📁 文件夹组织

### 推荐的文件夹结构：
```
📚 Obsidian使用指南/
├── 00-快速入门总览.md
├── 01-基础操作详解.md
├── 02-插件推荐.md
├── 03-高级技巧.md
└── 附件/  # 存储图片、PDF等附件
```

### 操作技巧：
- 在文件浏览器中右键文件夹/文件，可以进行重命名、移动、删除等操作
- 拖动文件到不同文件夹，可以移动笔记的存储位置（Obsidian会自动更新所有链接）

## ❓ 常见问题

1.  **Q：笔记丢失了怎么办？**
    A：Obsidian的所有笔记都存储在本地文件夹中，只要不删除原始文件，就可以通过「打开现有Vault」重新导入

2.  **Q：如何同步我的笔记？**
    A：可以使用Git、OneDrive、Dropbox等工具同步Vault文件夹，Obsidian官方也提供了同步服务

3.  **Q：可以离线使用吗？**
    A：完全可以！Obsidian是本地优先的，所有数据都存储在你的电脑上，不需要网络连接
