## 二分

```java 常规二分
public static int binarySearch(int[] nums, int target) {
    int left = 0;
    int right = nums.length - 1;

    while (left <= right) {
        // 防止溢出写法
        int mid = left + (right - left) / 2;

        if (nums[mid] == target) {
            return mid;         // 找到
        } else if (nums[mid] < target) {
            left = mid + 1;    // target 在右半部分
        } else {
            right = mid - 1;   // target 在左半部分
        }
    }

    return -1;  // 不存在
    return left;  //返回应该添加到的位置
}

```

```java 第一个>=、第一个>
class Solution {

    public int[] searchRange(int[] nums, int target) {
        int n = nums.length;
        int lefti = -1;
        int righti = -1;

        int left = 0, right = n-1;
        //找第一个 >= target 的位置
        while(left <= right) {
            int mid = left + ( (right - left) >> 1);
            if(nums[mid] >= target) right = mid-1;
            else left = mid + 1;
        }
        lefti = left;
        //找不到判断
        if (lefti == n || nums[lefti] != target) {
            return new int[]{-1, -1};
        }

        left = 0;
        right = n-1;
        //找第一个 > target的位置
        while(left <= right) {
            int mid = left + ( (right - left) >> 1);
            if(nums[mid] > target) right = mid-1;
            else left = mid + 1;
        }
        righti = left-1;    //因为是第一个> target的位置，所以-1

        return new int[]{lefti, righti};
    }
}
```

```java 找两个正序数组的第k小数
public int findKth(int[] a, int[] b, int k) {
    int i = 0, j = 0;

    while (true) {
        // 若某个数组已经用完，直接从另一个数组取第 k 个
        if (i == a.length) return b[j + k - 1];
        if (j == b.length) return a[i + k - 1];

        // k 变成 1，直接返回两个当前值中的较小者
        if (k == 1) return Math.min(a[i], b[j]);

        // 本轮尝试淘汰 k/2 个元素
        int half = k / 2;

        // 给 a 和 b 找各自的 pivot（越界则视为 +∞）
        int ni = Math.min(i + half, a.length) - 1;
        int nj = Math.min(j + half, b.length) - 1;
        int pa = a[ni];
        int pb = b[nj];

        // 淘汰 pivot 较小的那一段
        if (pa <= pb) {
            k -= (ni - i + 1);
            i = ni + 1;
        } else {
            k -= (nj - j + 1);
            j = nj + 1;
        }
    }
}

```

class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        int m = matrix.length, n = matrix[0].length;
        int i = 0, j = n-1;
        while(i < m && j >= 0) {
            if(matrix[i][j] > target) j--;
            else if(matrix[i][j] < target) i++;
            else return true;
        } 
        return false;
    }
}

## 动态规划

```java 01背包问题
class Solution {
    public int knapsack01(int[] weight, int[] value, int W) {
        int n = weight.length;

        // dp[j]：容量为 j 的背包，能获得的最大价值
        int[] dp = new int[W + 1];

        // 遍历每一个物品（0-1：每个物品只能用一次）
        for (int i = 0; i < n; i++) {
            // 容量必须倒序遍历，防止一个物品被重复使用
            for (int j = W; j >= weight[i]; j--) {
                // 不选当前物品 vs 选当前物品
                dp[j] = Math.max(
                    dp[j],                          // 不选
                    dp[j - weight[i]] + value[i]   // 选
                );
            }
        }

        return dp[W];
    }
}

```

```java 完全背包问题
class Solution {
    public int completeKnapsack(int[] weight, int[] value, int W) {
        int n = weight.length;

        // dp[j]：容量为 j 的背包，能获得的最大价值
        int[] dp = new int[W + 1];

        // 遍历每一个物品
        for (int i = 0; i < n; i++) {
            // 完全背包：容量正序遍历
            for (int j = weight[i]; j <= W; j++) {
                dp[j] = Math.max(
                    dp[j],                          // 不选当前物品
                    dp[j - weight[i]] + value[i]   // 选当前物品（可重复）
                );
            }
        }

        return dp[W];
    }
}

```

## 回溯算法

```java 全排列
class Solution {
    List<List<Integer>> res = new ArrayList<>();
    List<Integer> path = new ArrayList<>();
    boolean[] used;

    public List<List<Integer>> permute(int[] nums) {
        used = new boolean[nums.length];
        backtrack(nums);
        return res;
    }

    void backtrack(int[] nums) {
        if (path.size() == nums.length) {
            res.add(new ArrayList<>(path));
            return;
        }

        for (int i = 0; i < nums.length; i++) {
            if (used[i]) continue;   // 已经用过，跳过

            used[i] = true;          // 做选择
            path.add(nums[i]);

            backtrack(nums);         // 下一层继续选

            path.remove(path.size() - 1); // 撤销选择
            used[i] = false;
        }
    }
}

```

```java 组合
//从 1..n 中选 k 个元素的所有组合
class Solution {
    List<List<Integer>> res = new ArrayList<>();
    List<Integer> path = new ArrayList<>();

    public List<List<Integer>> combine(int n, int k) {
        backtrack(n, k, 1); // 从 1 开始选
        return res;
    }

    void backtrack(int n, int k, int start) {
        if (path.size() == k) {
            res.add(new ArrayList<>(path));
            return;
        }

        for (int i = start; i <= n; i++) {
            path.add(i);         // 选择
            backtrack(n, k, i+1);// 下一层从 i+1 开始
            path.remove(path.size()-1); // 撤销选择
        }
    }
}

```

```java 子集
class Solution {
    List<List<Integer>> res = new ArrayList<>();
    List<Integer> path = new ArrayList<>();

    public List<List<Integer>> subsets(int[] nums) {
        backtrack(nums, 0);  // 从下标 0 开始
        return res;
    }

    void backtrack(int[] nums, int start) {
        res.add(new ArrayList<>(path));  // 每一条路径都是一个子集

        for (int i = start; i < nums.length; i++) {
            path.add(nums[i]);         // 做选择
            backtrack(nums, i + 1);    // 递归，下一层从 i+1 开始
            path.remove(path.size() - 1); // 撤销选择
        }
    }
}

```

## 位运算

### 只出现了一次

```java
//除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。
class Solution {
    public int singleNumber(int[] nums) {
        int res = 0;
        for(int i : nums) {
            res ^= i;
        }
        return res;
    }
}
```

关于异或(^)的特性

1. a ^ a = 0
2. a ^ 0 = a
3. a ^ b ^ a = (a ^ a) ^ b = 0 ^ b = b

所以实际上这个代码的运行情况就是 0 ^ a ^ b ^ c ^ d ^......那么偶数次的就会被互相抵消，留下出现奇数次的数字
