## 推荐标准

这份清单优先选择三类工具：

- 免费可用，个人日常使用门槛低
- 在技术学习、开发协作或信息整理中有明确价值
- 社区活跃，文档相对完整，长期维护风险可控

-----------

## 开发效率类

### 1. Visual Studio Code

- 官网：<https://code.visualstudio.com/>
- 适合场景：代码编辑、轻量调试、多语言开发
- 推荐理由：插件生态成熟，启动和扩展速度都比较稳定

常用插件方向：

- 语言支持（Java、Python、Go、前端）
- Git 可视化增强
- Markdown 编辑与预览

### 2. GitHub Desktop

- 官网：<https://desktop.github.com/>
- 适合场景：本地 Git 提交、分支切换、冲突辅助处理
- 推荐理由：对 Git 命令不熟的阶段可显著降低操作失误率

### 3. Postman

- 官网：<https://www.postman.com/downloads/>
- 适合场景：接口调试、环境变量管理、请求集回放
- 推荐理由：学习后端接口和联调阶段上手快，团队共享也方便

-----------

## 文档与学习类

### 1. Obsidian

- 官网：<https://obsidian.md/>
- 适合场景：知识笔记、学习路径沉淀、双链整理
- 推荐理由：本地 Markdown 存储，迁移成本低，适合长期积累

### 2. Notion

- 官网：<https://www.notion.so/>
- 适合场景：项目计划、周报、文档协作
- 推荐理由：页面结构灵活，数据库视图对任务管理很实用

### 3. draw.io（diagrams.net）

- 官网：<https://www.diagrams.net/>
- 适合场景：架构图、流程图、系统设计草图
- 推荐理由：免费且功能完整，导出格式多，适配面广

-----------

## 调试与网络类

### 1. Apifox

- 官网：<https://apifox.com/>
- 适合场景：接口设计、调试、文档一体化管理
- 推荐理由：中文场景支持完整，个人项目从设计到调试链路顺畅

### 2. Wireshark

- 官网：<https://www.wireshark.org/>
- 适合场景：网络抓包、协议分析、问题定位
- 推荐理由：学习 TCP/IP、排查网络异常时证据链清晰

### 3. Everything（Windows）

- 官网：<https://www.voidtools.com/>
- 适合场景：本地文件秒级检索
- 推荐理由：排查日志、查历史文件时效率提升明显

-----------

## 云服务与部署辅助

### 1. Vercel

- 官网：<https://vercel.com/>
- 适合场景：前端项目快速部署、预览链接分享
- 推荐理由：个人项目上线路径短，自动化流程配置简单

### 2. Cloudflare

- 官网：<https://www.cloudflare.com/>
- 适合场景：DNS、CDN、HTTPS 与基础安全防护
- 推荐理由：免费层能力覆盖多数个人站点需求

### 3. GitHub Actions

- 官网：<https://github.com/features/actions>
- 适合场景：自动测试、自动构建、自动发布
- 推荐理由：与仓库集成度高，适合从手动发布过渡到 CI/CD

-----------

## 快速选型建议

| 目标 | 建议优先装 |
| :--- | :--- |
| 刚开始系统写代码 | VS Code + GitHub Desktop + Postman |
| 需要沉淀学习体系 | Obsidian + draw.io |
| 想做个人项目上线 | GitHub Actions + Vercel + Cloudflare |
| 排查接口与网络问题 | Apifox + Wireshark |

-----------

## 总结

工具本身不会替代技术能力，但会改变你处理问题的速度和稳定性。

先从一到两个高频场景入手建立固定工作流，再逐步补齐工具链，会比一次性安装大量工具更有效。
