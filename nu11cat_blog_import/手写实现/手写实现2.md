实现Trie前缀树，用于高效地存储和检索字符串数据集中的键。常见于自动补全和拼写检查。



<!-- more -->



灵感来自LeetCode [208. 实现 Trie (前缀树)](https://leetcode.cn/problems/implement-trie-prefix-tree/)

>Trie（发音类似 "try"）或者说 前缀树 是一种树形数据结构，用于高效地存储和检索字符串数据集中的键。这一数据结构有相当多的应用情景，例如自动补全和拼写检查。
>
>请你实现 Trie 类：
>
>- `Trie()` 初始化前缀树对象。
>- `void insert(String word)` 向前缀树中插入字符串 `word` 。
>- `boolean search(String word)` 如果字符串 `word` 在前缀树中，返回 `true`（即，在检索之前已经插入）；否则，返回 `false` 。
>- `boolean startsWith(String prefix)` 如果之前已经插入的字符串 `word` 的前缀之一为 `prefix` ，返回 `true` ；否则，返回 `false` 。

1. 都叫前缀树了，那么核心的数据结构肯定是树
2. 很容易想到每个节点一个字母，但是没办法判断字母是否已经存在。所以每个节点维护一个数组/哈希表来判断当前字母是否以及存在。另外需要注意的是，这里的数组一直都是初始化的，我们只通过它后面是不是null判断有无路径，只是作为一个索引表/路由表，没有其他含义。
3. 另外对于`search`和`startsWith`还需要每个节点有一个布尔类型表示当前节点是否是最后一个字母。
4. 大致的抽象表示如图(图片来自[LeetCode题解](https://leetcode.cn/problems/implement-trie-prefix-tree/solutions/98390/trie-tree-de-shi-xian-gua-he-chu-xue-zhe-by-huwt/?envType=study-plan-v2&envId=top-100-liked))：



![](../images/shouxieshixian/2_1.png)

所以就很好解决了，我们只需要每个节点设计为`int[26] + boolean` 。新增的时候遍历字符串，判断字符下是否有节点，没有就创建；search的时候查找 Trie 中是否存在给定前缀，并且这个节点是最后一个；startsWith就是查找 Trie 中是否存在给定前缀。

**代码：**

```java
class Trie {
    private Trie[] children;
    private boolean isEnd;

    public Trie() {
        children = new Trie[26];
        isEnd = false;
    }

    public void insert(String word) {
        Trie node = this;
        for(char i : word.toCharArray()) {
            int index = i - 'a';
            if(node.children[index] == null) {
                node.children[index] = new Trie();
            }
            node = node.children[index];
        }
        node.isEnd = true;
    }

    public boolean search(String word) {
        Trie node = searchPrefix(word);
        return node != null && node.isEnd;  //单词路径存在，并且该节点是单词的结尾
    }

    public boolean startsWith(String prefix) {
        return searchPrefix(prefix) != null;    //存在单词路径
    }

    // 查找 Trie 中是否存在给定前缀。
    private Trie searchPrefix(String prefix) {
        Trie node = this;
        for(char i : prefix.toCharArray()) {
            int index = i - 'a';
            if(node.children[index] == null) {
                return null;
            }
            node = node.children[index];
        }
        return node;
    }
}

 * boolean param_3 = obj.startsWith(prefix);
 */
```
