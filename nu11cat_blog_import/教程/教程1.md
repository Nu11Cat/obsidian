## Markdown



首先**Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。**



另外推荐Typora：它是一个 **Markdown 编辑器/写作软件**，它支持 Markdown 语法，并提供所见即所得的界面，让你写 Markdown 时可以直接看到排版效果。



附上官方链接：[Typora 官方中文站](https://typoraio.cn/)



关于`Markdown`的工作原理和应用不再赘述，可见官方文档



以下为语法的使用方式、最佳实现和注意事项。



## 基本语法



### 标题



很简单，

```text
# xxx就是一级标题
## 就是二级标题
```

#后面要有空格

更方便地，在 Typora 使用**CTRL+num**，可以直接使光标行变为 **num**级标题

另外，为了兼容，最好在标题前后都保留一个空行（不是空格，是回车那个空行）



### 粗体 



```text
**加粗的文字**
```

就能加粗你想要的文字

更方便地，在 Typora 使用**CTRL + B**直接把选中部分加粗，如果你选中的是加粗的文字那就是取消加粗



### 斜体



```text
*斜体的文字*
```

就能把你想要斜体的文字*倾斜*

更方便地，在 Typora 使用**CTRL + I**可以直接倾斜，效果同加粗



### 引用块



```text
>文字
```

效果：

> 这是引用块

你也可以

```
> Dorothy followed her through many of the beautiful rooms in her castle.
>
> The Witch bade her clean the pots and kettles and sweep the floor and keep the fire fed with wood.
```

>Dorothy followed her through many of the beautiful rooms in her castle.
>
>
>
>The Witch bade her clean the pots and kettles and sweep the floor and keep the fire fed with wood.

也可以

```
> Dorothy followed her through many of the beautiful rooms in her castle.
>
>> The Witch bade her clean the pots and kettles and sweep the floor and keep the fire fed with wood.
```

> Dorothy followed her through many of the beautiful rooms in her castle.
>
> > The Witch bade her clean the pots and kettles and sweep the floor and keep the fire fed with wood.

另外，引用块还能包含其他元素

```text
> #### The quarterly results look great!
>
> - Revenue was off the chart.
> - Profits were higher than ever.
>
>  *Everything* is going according to **plan**.
```

> #### The quarterly results look great!
>
> - Revenue was off the chart.
> - Profits were higher than ever.
>
>  *Everything* is going according to **plan**.

### 链接



许多markdown处理器会自动把URL转化成链接，你不希望这样的话可以使用反引号破坏链接

```

```

效果：[title](https://www.example.com)

也可以

```text
<http://markdownguide.cn>
```

<http://markdownguide.cn>

### 转义字符



如果你想要显示一个“原本用于markdown的文本字符”，可以加添一个反斜杠在前面。

如

```text
\* Without the backslash, this would be a bullet in an unordered list.
```

\* Without the backslash, this would be a bullet in an unordered list.

否则：

* Without the backslash, this would be a bullet in an unordered list.



### HTML

许多 Markdown 应用程序允许你在 Markdown 格式文本中使用 HTML 标签。

如：

```text
This **word** is bold. This <em>word</em> is italic.
```

This **word** is bold. This <em>word</em> is italic.


| 语法 | 说明 |

这么多‘|’和空格写起来太麻烦了，所以建议使用：

[Markdown Tables generator - TablesGenerator.com](https://www.tablesgenerator.com/markdown_tables)



**对齐**

你可以点击表格对整个表格的对齐方式进行指定，也可以通过下面这种方式针对某一列修改对齐方式

```text
| Syntax      | Description | Test Text     |
| :---        |    :----:   |          ---: |
| 左对齐      | 居中       | 右对齐   |
| Paragraph   | Text        | And more      |
```

| Syntax    | Description | Test Text |
| :-------- | :---------: | --------: |
| 左对齐    |    居中     |    右对齐 |
| Paragraph |    Text     |  And more |



另外，可以通过使用其 HTML 字符代码 (`|`)在表格中显示管道 (`|`) 字符。

```

```



**在表格内换行**可以通过HTML的\<br>实现

| 语法 | 描述                |

### 脚注



脚注允许你添加注释和参考

```text
这是一个带脚注的句子。[^1]
[^1]: 这是脚注。
```

比如：

其他内容....

[^1]: 这是脚注。



### 标题ID



```text
### 我的伟大标题 {#custom-id}
```

稍微解释下，这是一种自定义标题锚点的用法，常用于生成可以直接跳转的链接。

比如当你分享一个文章，maybe是:	xxx.com/文章/#我的伟大标题

但是当你使用了标题ID，你可以指定这个锚点的名字，可以是：	xxx.com/文章/#custom-id

允许你在文章中通过 `#custom-id` 跳转到这个标题的位置

仅此而已，不必想得太复杂

如果没有写 `{#custom-id}`，Markdown 会自动生成 ID（一般是把标题文字转小写、去掉空格）。

## 其他

### 下划线

一般用于撰写论文或报告

```
Some of these words <ins>will be underlined</ins>.
```

Some of these words <ins>will be underlined</ins>.

### 居中

md没有文本对齐概念，所以居中只能用HTML

```html
<center>This text is centered.</center>
```

<center>This text is centered.</center>

这个方式已经被官方弃用了，如果应用程序支持，可以使用：

```Html
<p style="text-align:center">Center this text</p>
```

<p style="text-align:center">Center this text</p>

### 颜色

Markdown不允许更改文本颜色，但是可以通过HTML实现

```html
<font color="red">This text is red!</font>
```

<font color="red">This text is red!</font>

但更建议：

```html
<p style="color:blue">Make this text blue.</p>
```

<p style="color:blue">Make this text blue.</p>

### 注释

其实我觉得更应该叫”隐藏文本“

效果是：作者可以看到文本，但不会将其打印在网页或 PDF 上

```
这是一个将显示的段落。

[这里是注释内容，将不会显示。]: # 

这是另一个将显示的段落。
```

这是一个将显示的段落。

[这里是注释内容，将不会显示。]: #


这是另一个将显示的段落。
